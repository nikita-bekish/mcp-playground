Model Context Protocol (MCP) — открытый протокол для соединения больших языковых моделей (LLM) с внешними инструментами, такими как API, базы данных и планировщики задач. MCP решает проблему отсутствия у моделей доступа к внешнему миру, обеспечивая динамическое предоставление ресурсов. Протокол работает поверх STDIO или WebSockets, что упрощает интеграцию в локальные агенты, редакторы, IDE и CI/CD. Инструменты объявляются сервером и автоматически доступны LLM, отличаясь от традиционных REST API по способу взаимодействия.
